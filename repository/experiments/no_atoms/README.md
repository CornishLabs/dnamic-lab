# No-atom experiments

These experiments characterise or configure laboratory equipment without using atoms.
They are deliberately separate from the atom-shot recipes in `../atoms/`.

Before submitting one, read its module docstring. Any experiment that requires a
physical intervention—such as inserting a power meter, changing a cable, blocking a
beam, or moving an optic—must include visible `Physical setup required` and
`Restoration required` sections there. Programmatic state which remains changed should
be documented under `Leaves hardware`.
